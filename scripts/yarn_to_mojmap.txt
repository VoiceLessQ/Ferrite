# Yarn -> Mojmap translation table for Ferrite 1.21.11 -> 26.1.2 port.
# Two sections:
#   [PACKAGES]   plain-string replacement on package qualifiers (works on imports
#                AND inline fully-qualified references like `net.minecraft.X.Y`)
#   [CLASSES]    word-boundary replacement (\b...\b) for renamed classes
# scripts/apply_mappings.sh consumes this file.
#
# Ordering rule: more-specific package prefixes BEFORE broader ones.  Critically,
# we do NOT use a broad `net.minecraft.world.` rule because its target
# `net.minecraft.world.level.` starts with the source string, which would re-fire
# on the output of every more-specific world.X -> world.level.X rule.  Instead
# the seven net.minecraft.world.<DirectClass> imports are listed explicitly.
